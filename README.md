# 🧠 MoodTracker

A beautiful, theme-aware journaling and mood tracking app built with **React**, **Tailwind CSS**, and **JavaScript**.  
Easily log your mood, jot down thoughts in a journal, and view motivational quotes — all in a clean, responsive UI.

![MoodTracker Preview](./preview.png) <!-- Optional: Add a screenshot here -->

---

## 🔮 Features

- 🌞 Dynamic greeting based on the time of day
- 📓 Journal entries to reflect and log your thoughts
- 😀 Mood tracker to record how you feel each day
- 💬 Daily motivational quotes (custom or from API)
- 🌙 Light/Dark mode with theme toggle
- 📱 Fully responsive and mobile-friendly

---

## ⚙️ Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **State Management**: useState, useEffect
- **Icons**: React Icons
